قالب مخصص لأجهزة الموبايل في ووردبريس:
لتحقيق ذلك في ووردبريس، نحتاج إلى استخدام دالة ذكية مدمجة في النظام تسمى wp_is_mobile(). هذه الدالة تتعرف تلقائياً على ما إذا كان الزائر يستخدم هاتفاً محمولاً أو جهازاً لوحياً.
أفضل طريقة لتطبيق هذا دون التعديل على ملفات القالب الأساسي (والتي قد تُحذف مع أي تحديث) هي إنشاء إضافة مخصصة (Custom Plugin).
1. كود الإضافة (Plugin Code)
قم بإنشاء ملف جديد وتسميته custom-mobile-theme.php وضع بداخله الكود التالي:






PHP
<?php
/**
* Plugin Name: Custom Mobile Theme Switcher
* Description: تغيير قالب الموقع تلقائياً عند التصفح من أجهزة الموبايل.
* Version: 1.0
* Author: مساعدك الذكي
*/

// حماية الملف من الوصول المباشر
if ( ! defined( 'ABSPATH' ) ) {
   exit;
}

function switch_theme_for_mobile( $template ) {
   // التحقق مما إذا كان الزائر يستخدم جهاز موبايل
   if ( wp_is_mobile() ) {
       // استبدل 'my-mobile-theme' بالاسم المجلد الصحيح لقالب الموبايل الخاص بك
       $mobile_theme = 'my-mobile-theme'; 
       
       return $mobile_theme;
   }
   
   // إذا كان جهاز كمبيوتر، اترك القالب الافتراضي كما هو
   return $template;
}

// ربط الدالة بالفلاتر المسؤولة عن تحديد القالب النشط في ووردبريس
add_filter( 'template', 'switch_theme_for_mobile' );
add_filter( 'stylesheet', 'switch_theme_for_mobile' );

2. شرح طريقة عمل الكود
يعتمد الكود على مفهوم الفلاتر (Filters) في ووردبريس لتعديل خيارات النظام قبل تحميل الصفحة:
* wp_is_mobile(): دالة تفحص "وكيل المستخدم" (User Agent) للمتصفح. إذا كان جهازاً محمولاً، تُرجع القيمة true.
* فلاتر template و stylesheet:
   * template: يحدد المجلد الذي يحتوي على ملفات القالب الأساسية (مثل index.php, header.php).
   * stylesheet: يحدد المجلد الذي يحتوي على ملف التصميم style.css (مهم جداً في حال كنت تستخدم قوالب الابن Child Themes).
* تغيير اسم القالب: عندما يكتشف الكود أن الزائر يستخدم الموبايل، يقوم باستبدال اسم القالب الحالي بالاسم المجلد الخاص بقالب الموبايل (my-mobile-theme).
3. خطوات التفعيل والتشغيل
1. تجهيز القوالب: تأكد من أنك تمتلك مجلدين لقوالب مختلفة داخل مسار wp-content/themes/. على سبيل المثال:
   * قالب سطح المكتب: parent-theme
   * قالب الموبايل: my-mobile-theme (يجب أن يطابق الاسم الموجود في الكود تماماً).
2. رفع الإضافة:
   * قم بضغط ملف custom-mobile-theme.php في ملف ZIP.
   * اذهب إلى لوحة تحكم ووردبريس -> إضافات -> أضف جديد -> رفع إضافة.
3. التفعيل: قم بتفعيل الإضافة.
⚠️ ملاحظة هامة جداً بخصوص الكاش (Caching):
إذا كنت تستخدم إضافات كاش مثل (WP Rocket, LiteSpeed Cache) أو كاش من جهة الخادم (مثل Cloudflare)، فقد يواجه الزوار مشكلة ظهور قالب الموبايل على سطح المكتب أو العكس.
الحل: يجب الدخول إلى إعدادات إضافة الكاش وتفعيل خيار "تخصيص كاش منفصل لأجهزة الموبايل" (Separate cache for mobile devices) لضمان عمل الكود بشكل سليم بنسبة 100%.
